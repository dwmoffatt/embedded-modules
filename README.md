This is a collection of embedded modules written in C/C++ for Arm Cortex-M microcontrollers.

## Architecture
```
├── arch
|   ├──
├── build
├── install
|   ├── install_deps.sh
└── projects
    ├── blinky_{mcu}
    .
    .
    .
```

## Project Requirements
All projects must have the below:
- Technical Requirements
- Schematics
- I/O Mapping
- Ability to Debug via Breakpoint
